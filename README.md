## 📋 نظرة عامة

نظام نقطة بيع (POS) وإدارة كامل مصمم خصيصاً للكافيهات. يدعم اللغة العربية بالكامل مع واجهة RTL احترافية. يعمل بدون إنترنت ويحفظ جميع البيانات محلياً.

## ✨ المميزات

### نقطة البيع (POS)
- واجهة سريعة لإضافة الطلبات وإدارتها
- تصفية الأصناف حسب القسم مع شريط أقسام قابل للتمرير
- بحث سريع عن الأصناف وترتيب حسب السعر
- سلة مشتريات تفاعلية مع تعديل الكميات
- دعم طرق دفع متعددة (كاش - بطاقة - فودافون كاش)

### سجل الطلبات
- عرض جميع الطلبات المكتملة مع تفاصيلها
- تعديل الطلبات (إضافة/حذف أصناف - تغيير الكمية)
- تعديل اسم العميل وطريقة الدفع وحالة الطلب
- حذف الطلبات مع تأكيد
- بحث وفلترة متقدمة

### إدارة القائمة
- إضافة وتعديل وحذف الأصناف
- تعديل الاسم والسعر والقسم لكل صنف
- 6 أقسام و 38 صنف جاهزين
- بحث وفلترة وترتيب

### إدارة العملاء
- إضافة وتعديل وحذف العملاء
- تصنيف العملاء (عادي / VIP)
- إنشاء طلب جديد مباشرة من صفحة العملاء
- إضافة تلقائية للعملاء الجدد عند إنشاء طلب

### إدارة الموظفين
- إضافة وتعديل وحذف الموظفين
- تحديد الأدوار (كاشير - ويتر)

### التقارير
- تقارير يومية بالمبيعات وعدد الطلبات
- المبيعات حسب القسم (رسوم بيانية)
- أداء الموظفين
- تصدير PDF بالعربي عبر الطباعة

### لوحة التحكم
- إحصائيات سريعة (المبيعات - الطلبات - العملاء)
- جدول بآخر العملاء المسجلين
- آخر الطلبات المكتملة

### الأمان
- تسجيل دخول محمي بكلمة مرور
- تغيير كلمة المرور من داخل التطبيق

## 🚀 التشغيل

### المتطلبات
- [Node.js](https://nodejs.org/) v18 أو أحدث

### التثبيت

```bash
# استنساخ المشروع
git clone https://github.com/Ezatnasef/-Cafe.git
cd -Cafe

# تثبيت الحزم
npm install

# تشغيل في وضع التطوير
npm run dev
```

### البناء والتشغيل

```bash
# بناء المشروع
npm run build

# تشغيل النسخة النهائية
npm start
```

## 📁 هيكل المشروع

```
├── client/                 # واجهة المستخدم (React)
│   ├── src/
│   │   ├── components/     # المكونات المشتركة
│   │   ├── hooks/          # React Hooks مخصصة
│   │   ├── pages/          # صفحات التطبيق
│   │   └── lib/            # أدوات مساعدة
│   └── index.html
├── server/                 # الخادم الخلفي (Express)
│   ├── routes.ts           # نقاط API
│   ├── storage.ts          # طبقة التخزين (JSON)
│   └── index.ts            # نقطة البدء
├── shared/                 # كود مشترك
│   ├── schema.ts           # Zod schemas و TypeScript types
│   └── routes.ts           # عقود API
└── data/                   # قاعدة البيانات المحلية (يتم إنشاؤها تلقائياً)
    └── db.json
```
